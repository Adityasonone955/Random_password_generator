# 🔐 Random Password Generator
A simple and user-friendly web-based Random Password Generator that allows users to create secure passwords based on their preferences. Users can select the desired password length and customize the composition by including uppercase letters, lowercase letters, numbers, and special symbols.

This project focuses on improving password security by generating strong, unpredictable combinations, helping users protect their accounts and sensitive data.


# 🚀 Features
Adjustable password length
Option to include:
Uppercase letters (A–Z)
Lowercase letters (a–z)
Numbers (0–9)
Special characters (!@#$%^&*)
Instant password generation
Simple and responsive UI
# 🛠️ Tech Stack
HTML
CSS
JavaScript

# 🎯 Purpose
The main goal of this project is to demonstrate the implementation of randomization logic, user input handling, and dynamic UI updates using core web technologies.
